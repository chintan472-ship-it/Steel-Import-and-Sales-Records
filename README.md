Project Overview :

This project focuses on analyzing steel import operations and sales performance using Power BI and Excel.

The goal is to transform raw business data into actionable insights that help management understand:

Supplier pricing differences

Regional sales performance

Product profitability

Monthly sales trends

Overall business performance

The dashboard helps stakeholders make data-driven decisions related to procurement, pricing strategy, and sales planning.

Business Problem

Steel trading companies deal with large volumes of import shipment data, supplier pricing, and sales records.

Without structured reporting, it becomes difficult to:

Track supplier cost efficiency

Monitor sales performance across regions

Identify high-profit products

Understand sales trends over time

This project solves these problems by creating an interactive analytics dashboard.

Tools & Technologies

Power BI – Data visualization and dashboard creation

Microsoft Excel – Data cleaning and transformation

Power Query – Data preparation and shaping

DAX – Creating business KPIs and calculated metrics

Key Insights Generated

The dashboard helps identify:

Top-performing products contributing the highest revenue

Most profitable product categories

Supplier cost differences

Monthly sales fluctuations

These insights support better procurement and pricing decisions.
